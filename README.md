# VueTodoProject

뷰로 만든 Todo 프로젝트입니다.

## 설명

Vue로 제작한 Todo List 프로젝트입니다.
현재 오리지널 Vue.js로 제작한 상태이며, ES6 구문과 vueX 구문을 배우면 리펙토링할 예정입니다.
