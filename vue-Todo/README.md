# VueTodoProject

뷰로 만든 Todo 프로젝트입니다.

## 설명

Vue로 제작한 Todo List 프로젝트입니다. <br>
현재는 localStroage에 데이터를 저장해 Todo를 띄우는 방식을 채택하고 있습니다.
나중에 MongoDB 또는 FireBase로 변경하여 수정 기능까지 완성시킬 계획입니다.
